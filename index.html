BrickStat

In Progress

When building a Lego set, you may wonder how much of a time commitment a set is going to take before you build it. BrickStat allows you to input designated factors asked by the site to give the user an estimated time it is going to take to build the set.

Highlights

  • Utalizes API calls to Rebrickable for lego set data.
  • Features an innovative user interface that takes user input to give time estimates.
  • Includes a user review system so people can see how long it took them to build the set, using the same factors the time estimation uses.
